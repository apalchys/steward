# Summary

Introduce reusable settings cards, rows, editor cards, and supporting layout primitives.

# Scope

- Add reusable view-only building blocks for grouped settings presentation.
- Use rounded cards, thin strokes, inset dividers, and native control alignment.
- Reuse the new primitives across general, feature, clipboard, and about panes.

# Acceptance Criteria

- Settings content is composed from reusable card and row components.
- Rows present leading title and description with trailing control when applicable.
- Multi-row groups use inset separators and consistent padding.
